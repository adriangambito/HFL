# HFL
A Hierarchical Federated Learning reproducible simulation


**Installation requirments:**
*	Flower 1.20.0

```bash
git clone https://github.com/adriangambito/HFL.git
cd HFL

python3 -m venv hfl-env
source hfl-env/bin/activate

pip install '-r requirements.txt'
```


**Launch the application**
```
python3 main.py
```
